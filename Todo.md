# Todo

_Task format: `- [ ] Task description [[Project Name]]` - project link optional, used when the task ties to a project note._
_Deadline format: append `` `due:YYYY-MM-DD` `` for real external commitments only._
_Completed format: `- [x] Task` with `` `done:YYYY-MM-DD` ``, moved to Done section._

## Inbox
_New tasks land here. Triaged weekly._

## This Week
_Active tasks for the current week. Promoted from Inbox during weekly planning._

## Backlog
_Tasks that matter but aren't this week._

## Someday / Maybe
_Ideas and low-priority items. Revisit quarterly._

## Done
_Completed tasks. Cleared during weekly review._
